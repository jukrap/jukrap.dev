# Planning Docs

These public notes support the 2026 portfolio modernization. Current agent
memory, worklogs, and archived handoffs live in `.ai-playbook/` when that local
folder exists.

Read these files in this order when working on the modernization:

1. `AGENTS.md`
2. `.ai-playbook/START_HERE.md` and `.ai-playbook/CURRENT.md` when present
3. `PROJECT_SPEC.md`
4. `PLANS.md`
5. `docs/plans/2026-06-02-portfolio-modernization-analysis.md`

## Roles

- `AGENTS.md`: thin agent bootstrap and source-of-truth rules.
- `.ai-playbook/`: local project memory, maps, runbooks, active plans, worklogs, and archived AI notes.
- `PROJECT_SPEC.md`: product goal, content boundaries, and feature scope.
- `PLANS.md`: milestone criteria and verification commands.
- `docs/plans/**`: public analysis, implementation notes, and future handoffs.
